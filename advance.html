Advance commands

serial no. commands Discription
1 ifconfig ifconfig is used to configure the kernel-resident network interfaces.
2 netstat -a netstat command displays various network related information such as network connections, routing tables, interface statistics, masquerade connections, multicast memberships etc..,
3 dig dig is a tool for querying DNS nameservers for information about host addresses, mail exchanges, nameservers, and related information. This tool can be used from any Linux (Unix) or Macintosh OS X operating system. The most typical use of dig is to simply query a single host.
4 uptime You have just connected to your Linux Server Machine and founds Something unusual or malicious,you could run uptime to verify what happened actually when the server was unattended.
5 mesg Lets you control if people can use the “write” command, to send text to you over the screen.
6 write Let you send text directly to the screen of another Linux machine if ‘mesg’ is ‘y’.
7 rename this command rename files. rename will rename the specified files by replacing the first occurrence from the file name.
8 top Displays the processes of CPU. This command refresh automatically, by default and continues to show CPU processes unless interrupt-instruction is given.
9 free Keeping track of memory and resources is as much important, as any other task performed by an administrator, and ‘free‘ command comes to rescue here.
10 lsof lsof stands for “list open files” and displays all the files that your system has currently opened.
