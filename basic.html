Basic commands

serial no. commands Discription
1 date find date
2 cal find calender
3 whoami find whoamii
4 copy use for copy any file
5 move use for move a file
6 list show all of the major directories filed under a given file system
7 change directory The cd command - change directory - will allow the user to change between file directories
8 clear the clear command clears the screen and wipes the board clean
9 mkdir The mkdir - make directory - command allows the user to make a new directory. Just like making a new directory within a PC or Mac desktop environment, the mkdir command makes new directories in a Linux environment.
